# 🏨 Hotel Booking Cancellation Prediction

## 📖 Overview

This project predicts whether a hotel booking will be canceled using machine learning techniques. Booking cancellations significantly impact hotel revenue and planning, so this model helps identify high-risk bookings in advance.

---

## 🎯 Objectives

* Predict booking cancellations (Canceled / Not Canceled)
* Perform exploratory data analysis (EDA)
* Build and evaluate machine learning models
* Identify key factors influencing cancellations

---

## 📊 Dataset

The dataset contains hotel booking information such as:

* Lead time (days between booking and arrival)
* Number of guests (adults, children)
* Length of stay
* Room type and meal plan
* Average Daily Rate (ADR)
* Special requests

---

## 🔍 Exploratory Data Analysis

* Distribution of numerical and categorical features
* Missing value analysis
* Relationship between features and cancellation status
* Class imbalance detection

---

## ⚙️ Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Feature scaling
* Train-test split

---

## 🤖 Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* K-Nearest Neighbors (KNN)

---

## 📈 Model Evaluation

Models are evaluated using:

* Accuracy
* Precision
* Recall
* F1-score

---

## 💡 Key Insights

* Longer lead time increases cancellation probability
* Pricing and booking conditions affect cancellations
* Special requests reduce cancellation likelihood
* Feature engineering improves performance

---

## 🚀 Applications

* Revenue management
* Overbooking strategies
* Customer behavior analysis
* Risk management in bookings

---

## 🛠️ Tech Stack

* Python

---

## 📌 Conclusion

This project shows how machine learning can help hotels reduce losses by predicting booking cancellations and enabling data-driven decisions.
